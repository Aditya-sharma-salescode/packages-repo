# @aditya-sharma-salescode/shared-ui

Shared UI primitives and utilities used across `@aditya-sharma-salescode/*` packages.

## Install

```bash
npm i @aditya-sharma-salescode/shared-ui
```

## Peer dependencies

Your app must provide:
- `react`
- `react-dom`

## Usage

```ts
import { cn, Button } from "@aditya-sharma-salescode/shared-ui";
```

## Notes

- Components are exported from `src/index.ts` and built to `dist/` via `tsup`.
